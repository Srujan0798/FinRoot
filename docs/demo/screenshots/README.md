## How to regenerate

```bash
pip install playwright && playwright install chromium
PYTHONPATH=src python3 scripts/capture_screenshots.py
```

The script requires Chromium (via Playwright). All screenshots are rendered in
Mock mode (`FINROOT_LLM_PROVIDER=mock`) — no API keys needed.
