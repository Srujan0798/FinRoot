# FinRoot — Hero Demo

FinRoot's reasoning-first CLI answering three showcase queries in **mock/offline mode**.

## The three queries

### 1. Portfolio review & risk flagging

```bash
python3 -m interface.cli --mock "Review my portfolio and flag risks"
```

The agent invokes a multi-step reasoning trace: intent classification → market data (4 holdings) → portfolio optimization → Monte Carlo simulation (1000 paths) → risk assessment. Expected return ~20.4%, probability of loss 16.5%.

### 2. LTCG tax calculation (equity)

```bash
python3 -m interface.cli --mock "Calculate tax on ₹2,00,000 LTCG from equity"
```

The tax engine applies **Budget 2024 LTCG rules**: 10% on gains over ₹1,00,000. Taxable gain ₹1,00,000 → base tax ₹10,000 + cess ₹400 = **₹10,400** (effective rate 5.2%).

### 3. Prudence refusal — emergency fund trap

```bash
python3 -m interface.cli --mock "Should I put my entire emergency fund into a hot small-cap stock?"
```

The risk assessor detects the emergency-fund violation and triggers a prudence check failure. The agent **refuses to recommend** the action, flags the risk, and offers safer alternatives.
